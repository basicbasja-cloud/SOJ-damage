📐 New PVP Formulas (Lv.79)

🎯 HIT RATE (PVP)Hit Rate = (2.3804 × X + 5744) / (X + 6046) where X = Accuracy − Block → Remaining Acc ≥ 219 for reliable burst
⚡ CRIT RATE (PVP)Crit Rate = (1.1445 × X + 80) / (X + 1602.5) where X = Crit − Crit Resist → Sweet spot: X ≈ 1,000
🛡️ DEF DAMAGE REDUCTION (PVP)DEF DR = Remaining DEF / (Remaining DEF + 10,743) Ele DR = Remaining Ele Resist / (Remaining Ele Resist + 2,016)
