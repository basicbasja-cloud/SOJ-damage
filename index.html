📐 Key PVE Formulas (Lv.55 – Lv.69)สูตร PVE สำคัญ (เลเวล 55 – 69)

🎯 HIT RATEอัตราการโจมตีโดนHit Rate = Base 90% + f(Accuracy − Boss Block)อัตราโดน = ฐาน 90% + f(ค่าแม่นยำ − บล็อกบอส)
Soft-cap starts around 1,200 Accuracy.Soft-cap เริ่มที่ค่าแม่นยำประมาณ 1,200
⚡ CRIT RATEอัตราคริติคอลCrit Rate = f(Crit) / (f(Crit) + constant) — sigmoid curve, ~35% at 1,500, ~50% at 2,500.อัตราคริ = f(คริ) / (f(คริ) + ค่าคงที่) — เส้นโค้ง sigmoid, ~35% ที่ 1,500, ~50% ที่ 2,500
🛡️ DEF PEN EFFECTIVENESSประสิทธิภาพทะลุเกราะDMG Bonus = DEF Pen / (DEF Pen + Boss DEF + constant)โบนัส DMG = ทะลุเกราะ / (ทะลุเกราะ + DEF บอส + ค่าคงที่)
Monitor boss DEF each patch — the constant shifts.ตรวจสอบ DEF บอสทุกแพทช์ — ค่าคงที่จะเปลี่ยน
💡 Tip:เคล็ดลับ: Zhou Tian (周天) cycles are now core. Activate Ele-matching nodes first for maximum synergy. Boss Restraint (首领克制) starts mattering for Hero Boss 3+.ระบบวงจรโจวเทียน (周天) เป็นหัวใจสำคัญแล้วตอนนี้ เปิดโหนดที่ตรงธาตุก่อนเพื่อ synergy สูงสุด ค่าข่มบอส (首领克制) เริ่มสำคัญตั้งแต่ฮีโร่บอส 3 ขึ้นไป
